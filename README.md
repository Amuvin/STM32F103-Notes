<h1 align="center"> 📒 关于STM32F103学习笔记</h1>

## 📃 仓库介绍

**记录和分享个人的STM32F103学习笔记**

* C语言，STM32标准库
* 技术栈有GPIO，中断，TIM，UART，IIC，SPI，PWM，ADC，DMA
* Keil5，VScode开发平台
* 本仓库精简了很多知识点，大多数以**图文形式**简单表达，方便读者快速配置
* 笔记为**Makdown**语法文档，可进行二次编辑

## 💪 适用人群

* 之前学习过STM32想**快速复习**知识点的
* 初学者想知道各个功能是怎么配置实现的

## 😲 注意！！

- 本文面向的是有**简单接触过单片机**或者想**快速复习**的的初学者
